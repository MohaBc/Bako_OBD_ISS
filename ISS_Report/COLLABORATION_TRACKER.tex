\section*{Guidelines \& Best Practices}

\begin{enumerate}
    \item Always create a feature branch for section work
    \item Test compilation locally before pushing
    \item Write clear, descriptive commit messages
    \item Follow LaTeX formatting conventions (see README.md)
    \item Keep section files independent and well-organized
    \item Review README.md for detailed editing guidelines
    \item Test the PDF output carefully after changes
    \item Use proper cross-references with \texttt{\textbackslash label\{\}} and \texttt{\textbackslash ref\{\}}
\end{enumerate}

\section*{Quick Reference: Section Checklist}

Before submitting your section for review, ensure:

\begin{enumerate}
    \item[\checkmark] Placeholder text replaced with actual content
    \item[\checkmark] All subsections have substantial content
    \item[\checkmark] LaTeX compiles without errors
    \item[\checkmark] No undefined references or citations
    \item[\checkmark] Images properly referenced and located
    \item[\checkmark] Spelling and grammar reviewed
    \item[\checkmark] Formatting consistent with other sections
    \item[\checkmark] Cross-references use proper labels
    \item[\checkmark] Bibliography entries correct in \texttt{references.bib}
    \item[\checkmark] PDF output looks professional and complete
\end{enumerate}
